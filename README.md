***PROYECTO FINAL***
Para el proyecto final se nos pidió realizar en verilog un procesador MIPS de 32 bit. Comenzamos el semetre viendo las bases como cración de módulos cada vez más y más complejos, para terminar realizando el proyecto. Esta consta basicamente de 3 etapas.
Fase 1. Implementación de módulos para implementar las instrucciones tipo R (Register)
Fase 2: Implementación de módulos para implementar instrucciones tipo I (Immediate)
Fase 3: Implementacioón de modulos 
Se dividió de la siguiente manera: 
La carpeta "Proyecto Final Arquitectura de Computadoras" es la que fue usada durante el semestre para subir cada una de elas 3 fases que comprendía la realizacion de los modulos para implementación de instrucciones Tipo R, tipo I y tipo J en el procesador MIPS de 32 bits realizado con el lenguaje de descipcion de hardware: Verilog. Además, se hizo un reporte por cada fase. Cabe aclarar que, al ser etsa la carpeta donde se guardaban a su vez subcarpetas de cada una de las fases, fueron estas y sus enlaces las que fueron proveídas para el profesor en classroom. 
La carpeta "Power Rangers - Proyecto final - Arquitectura de Computadoras" es la entrega final y unificada de todo el proyecto. De cierta manera es igual a la anterior pues tambien contiene las 3 fases pero, a consideración de que en esta carpeta se tiene mas organizado la información en 4 subcarpetas que son: Docs, images, RTL y src. 

Nota: La razón por la cual existen dos carpetas para el proyecto, cuando una contiene lo de la otra y más, es que, la llamada "Proyecto Final Arquitectura de Computadoras" fue la que compartí al profesor durante el semestre, no puedo borrarla por el momento o quedaría como carpeta inexistente, lo cual podría afectar.
